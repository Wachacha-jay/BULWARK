You are a senior frontend architect and product designer. Your task is to design and implement a modern, high-performance, scalable frontend for a construction materials company (Bulwark Industries, Kenya).

The frontend must not be a simple marketing website. It must function as a **conversion engine, decision-support interface, and data collection layer**, fully integrated with backend APIs and future ML systems.

---

## CORE OBJECTIVES

1. Maximize conversion (turn visitors into customers or distributors)
2. Provide an intuitive, frictionless user experience
3. Integrate seamlessly with backend APIs (FastAPI)
4. Support SEO and content marketing
5. Be scalable and maintainable long-term

---

## TECH STACK

* Framework: Next.js (latest version, App Router)
* Language: TypeScript
* Styling: Tailwind CSS
* State management: React Query / Zustand
* Forms: React Hook Form + Zod validation
* SEO: Next.js metadata API
* Analytics hooks ready (GA4, custom tracking)

---

## INFORMATION ARCHITECTURE

Design the site with clear navigation:

1. Home
2. Products
3. How It Works
4. For Contractors / Developers (B2B)
5. Distributor Program
6. Blog / Resources
7. Contact / Support

---

## KEY FEATURES TO IMPLEMENT

1. Smart Quote Calculator (Frontend)

* Input: project size (sqm)
* Call backend API: /quote
* Display:

  * Bags required
  * Estimated cost
* Must be fast, interactive, and mobile-friendly

2. Product Recommendation Tool

* Inputs:

  * Surface type
  * Climate
  * Project size
* Call backend API: /recommend
* Display structured recommendation

3. WhatsApp Integration

* Floating WhatsApp button
* Pre-filled quick messages (pricing, coverage, distributors)

4. Multi-CTA System

* “Request Quote”
* “Become a Distributor”
* “Download Datasheet”
* “Contact Sales”
